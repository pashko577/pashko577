### 🛒 **ProyPeruMarket – Sistema de Gestión Comercial**

> *Plataforma integral para la gestión de operaciones comerciales en pequeñas y medianas empresas peruanas*

<div align="center">

![React](https://img.shields.io/badge/React-18-61DAFB?style=flat-square&logo=react&logoColor=black)
![Vite](https://img.shields.io/badge/Vite-5.x-646CFF?style=flat-square&logo=vite&logoColor=white)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-3.x-06B6D4?style=flat-square&logo=tailwindcss&logoColor=white)
![Java](https://img.shields.io/badge/Java-17-ED8B00?style=flat-square&logo=openjdk&logoColor=white)

![MySQL](https://img.shields.io/badge/MySQL-8.0-4479A1?style=flat-square&logo=mysql&logoColor=white)

</div>

| Característica | Detalle |
|----------------|---------|
| **Frontend** | React 18 + Vite + TailwindCSS |
| **Backend** | Java 17, Spring Boot 3.x, JPA/Hibernate |
| **Base de Datos** | MySQL |
| **Seguridad** | JWT, Roles (Administrador, Empleado) |
| **Arquitectura** | Clean Architecture con separación de capas |

**🎯 Objetivo del Proyecto:**
Optimizar los procesos de registro de productos, control de ventas, gestión de usuarios y generación de reportes para mejorar la organización y toma de decisiones del negocio.

**🧩 Problema que Resuelve:**
Muchas pequeñas empresas manejan sus ventas de forma manual (Excel o cuadernos), sin control de stock en tiempo real ni reportes confiables. ProyPeruMarket digitaliza estos procesos, centralizando toda la información en un sistema.

**⚙️ Funcionalidades Principales:**

| Módulo | Funcionalidades | Tecnología |
|--------|-----------------|------------|
| 👤 **Gestión de Usuarios** | Registro, Login, Roles, JWT | Spring Security + React Context |
| 📦 **Gestión de Productos** | CRUD, Stock real-time, Categorías | JPA + React Query |
| 💰 **Gestión de Ventas** | Registro, Cálculo automático, Historial | Transactions + Zustand |
| 📊 **Reportes** | Gráficos interactivos, Exportación | Chart.js + Tailwind |
